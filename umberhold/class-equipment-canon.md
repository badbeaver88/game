# Umberhold Class Hit Dice and Equipment Canon

This document captures the current approved rules for:
- class hit dice per level
- armor and shield restrictions
- weapon proficiencies
- non-proficient weapon use

It does **not** modify `MASTER.md`.

---

## 2. Hit Points Per Level

A character's hit points are determined by their class hit die.

### Hit Point Roll Rule
At each level:
- roll the class hit die
- add the character's **Constitution modifier** to the roll

### No Average Option
There is **no average hit point option**.
Every level requires an actual hit point roll.

---

## 3. Armor Restrictions

Armor restrictions are **absolute**.

If a class cannot equip a given armor type, it cannot wear that armor under any circumstances.

### Inventory Handling
A character may still carry unusable armor, but it should be displayed in **red text** in the inventory to show that it cannot be equipped by that class.

---

## 4. Shield Restrictions

Shield restrictions follow the class table exactly.

If a class cannot equip a shield, it cannot use one under any circumstances.

---

## 5. Weapon Proficiency Rules

Any class may equip any weapon.

However, the **Proficiency Bonus** is only applied to attack rolls when the class is proficient with that weapon.

### Non-Proficient Weapon Rule
If a class uses a weapon it is not proficient with:
- the attack may still be made
- the attack roll does **not** add the Proficiency Bonus

---

## 6. Approved Rules Captured Here

This document locks in the following:
- each class has a defined hit die per level
- Constitution modifier is added to each hit point roll
- every level requires a hit point roll
- there is no average hit point option
- armor restrictions are absolute
- shield restrictions are absolute
- unusable armor may still be carried but should display in red in inventory
- any class may equip any weapon
- weapon proficiency only affects whether Proficiency Bonus is added to attack rolls
- non-proficient weapon attacks do not gain Proficiency Bonus

---

## 7. Still Open for Later Canonization

The following are still open and may need later clarification:
- whether first-level characters still begin with maximum hit points from their class hit die
- whether off-hand weapon proficiency follows the same rule with no exceptions
- whether class proficiencies can ever be expanded by training, items, or special features
- whether instruments for Troubadours count as weapon-category items for equipment UI purposes
- whether some relic or class effects can temporarily bypass armor restrictions
