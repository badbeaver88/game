## Biggest Spell Gaps

### Hedge Wizard
Has the most visible spell identity so far, but still needs:
- 2nd-5th level spell library
- more battlefield control
- more AoE options
- more utility/exploration spells

### Friar
Needs far more content.
Currently only clearly represented by:
- Cure Light Wounds

Still needs:
- protection spells
- status cleansing
- anti-undead magic
- blessings/buffs
- higher-level recovery miracles

### Crusader
Needs far more content.
Currently only clearly represented by:
- Cure Light Wounds access

Still needs:
- holy strike / smite-style effects
- self/ally support
- front-rank buff tools
- limited divine offense

### Warden
Has:
- Cure Light Wounds
- Witchlight
- Warden’s Mark

Still needs:
- more hunt/mark/tracking identity
- archery support magic
- nature/survival utility if intended
- 2nd-3rd level spell coverage

### Troubadour
Has:
- Light
- Mocking Words
- Witchlight

Still needs:
- song-like support spells
- morale effects
- crowd control/debuff support
- out-of-combat party utility
- 2nd-3rd level spell coverage
