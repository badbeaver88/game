# Umberhold

Design documents and rules for the Umberhold project.

---

## Directory Structure

| Directory / File | Purpose |
|---|---|
| `DESIGN.md` | Master design document |
| `plan.md` | Development plan and milestones |
| `gap.md` | Known gaps and open questions |
| `list.md` | Feature and content checklists |
| `item-list.md` | Consolidated item reference table |
| `classes/` | Individual class definitions (one file per class) |
| `content/` | Narrative and hub content (village, NPC dialogue) |
| `examples/` | Example data files for schemas |
| `items/` | Individual item entries and item tables |
| `meta/` | Content metadata extractions and gap analysis |
| `monsters/` | Individual monster stat blocks |
| `races/` | Individual race definitions (one file per race) |
| `rules/` | Canonical rules documents (combat, conditions, currency, etc.) |
| `schemas/` | Machine-readable data schemas |
| `spells/` | Spell definitions and class spell lists |
| `templates/` | Templates for authoring new content |
