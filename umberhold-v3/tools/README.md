# Tools

Authoring and support tooling used alongside the Umberhold design docs.

## Included

- `gridmapper/` — a vendored, browser-based dungeon drafting tool adapted to Umberhold's `26x26` map size.
- `research/3d-dungeon-review.md` — review notes for the `uheartbeast/3d-dungeon` Godot prototype as a future exploration/rendering reference.
