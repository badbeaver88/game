## Race Index

|---|---|---|

| [Everkin](./everkin.md) | Medium | +1 AGI, +1 INT; bow accuracy, bow crit, sleep/charm resistance |
| [Deepfolk](./deepfolk.md) | Medium | +1 CON, +1 STR; +1 HP/level, poison save bonus |
| [Stoutlings](./stoutlings.md) | Small | +1 LCK, +1 AGI; +4 lockpicking, +2 all saves |
| [Tinkerfolk](./tinkerfolk.md) | Small | +1 INT, +1 LCK; magic item crit doubling on nat 20 |
| [Hillkin](./hillkin.md) | Large | +1 CON, +1 STR; Base AC 11, -4 AGI saves |
